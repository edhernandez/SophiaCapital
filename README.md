# Sophia Capital

Asesoría financiera profesional para empresas y personas.

## Stack Tecnológico

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Radix UI

## Desarrollo Local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Build de Producción

```bash
npm run build
npm start
```
